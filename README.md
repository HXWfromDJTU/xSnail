# xSnail

```
xSnail is a mixup SPA for ON Studio 
```

## forntend : vue2 + webapck + es6 + fetch + vue-router + Vant 
frontend start up
```
1. $ npm install
2. $ npm run dev
```

## backend: egg + es6 
    Using Egg framework as backend providing the data source API
backend start up
```
1. $ cd backend
2. $ npm run dev 
```

## storage : mysql + redis
sql startup
```
1. mysql db name xSnail
2. $ redis-server
```
