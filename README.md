# xSnail
## xSnail is a web app 
## forntend : vue2 + webapck + es6 + fetch + Vant 
## bakcend: egg + es6 
## storage : mysql + redis

### frontend start up
```
1. $ npm install
2. $ npm run dev
```


### backend start up
```
1. $ cd backend
2. $ npm run dev 
```

### sql startup
```
1. mysql db name xSnail
2. $ redis-server
```
